Album
=====

Retrieve Album
--------------

.. http:get:: /album/(string:album_id)
    :noindex:
    :synopsis: Retrieve information about an album

    **Example request**:

    .. sourcecode:: http

        GET /album/ab6d4b6287fc HTTP/1.1
        Host: example.com
        Accept: application/json, text/javascript
