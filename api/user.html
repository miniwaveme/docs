User¶

Retrieve user¶

GET /user/get/(string: user_id)

Example request:

GET /track/get/ab6d4b6287fc HTTP/1.1
Host: example.com
Accept: application/json, text/javascript

Example response:

HTTP/1.1 200 OK
Vary: Accept
Content-Type: text/javascript

{
    "id": "ab6d4b6287fc",
    "nickname": "mhor",
    "email": "maxime.horcholle@gmail.com",
    "gender": "male",
    "birthdate": "1987-05-18",
    "firstname": "maxime",
    "lastname": "horcholle"
}
Query Parameters:
 
  • user_id – user id
Status Codes:
  • 200 OK – no error
  • 404 Not Found – there’s no user
