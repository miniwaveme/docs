Recommendation¶

Get recommendations for an user¶

GET /user/(string: user_id)/recommendation/list

Example request:

GET /user/ab6d4b6287fc/recommendation/list HTTP/1.1
Host: example.com
Accept: application/json, text/javascript

Example response:

HTTP/1.1 200 OK
Vary: Accept
Content-Type: text/javascript

[
