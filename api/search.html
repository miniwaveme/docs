Search¶

Search track¶

GET /search/track

Example request:

GET /search/track HTTP/1.1
Host: example.com
Accept: application/json, text/javascript

Example response:

HTTP/1.1 200 OK
Vary: Accept
Content-Type: text/javascript

{
    "result_count": 120,
    "total_page": 3,
    "page": 1,
    "result": [
        {
            "type": "track",
            "id": "",
        },
        {
            "type": "track",
            "id": "",
        },
        {
            "type": "track",
            "id": "",
        }
    ]
}
Status Codes:
  • 200 OK – no error
